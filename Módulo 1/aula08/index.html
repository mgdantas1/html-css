Trechos de códigos

Para utilizarmos trechos de códigos nos nossos sites utilizamos a tag <code></code>, entretanto, ela deixa a letra mono-espaçada apenas. Para que o texto fique com as tabulações, quebras de linhas e espaços utilizamos a tag <pre></pre>.

        
            num = int(input('Digite um número: '))

            if num % 2 == 0:
                print('PAR!')
