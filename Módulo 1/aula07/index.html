Principais formatações

Itálico e Negrito

Para colocar algo em Negrito utilizamos as tags <strong></strong>.

Para colocar algo em Itálico utilizamos as tags <em></em>.

Marca texto

Para marcar o texto utilizamos a tag <mark></mark>.

Textos grandes e pequenos

Para textos grandes temos a tag <big></big>, mas ela está entrando em desuso.

Para textos pequenos temos a tag <small></small>.

Texto deletado

Para um texto que deve ser desconsiderado utilizamos a tag <del></del>.

Texto inserido

Para textos inseridos utilizamos a tag <ins></ins>.

Textos sobrescritos e subscritos

Para textos sobrescritos (X2) temos a tag <sup></sup>.

Para textos subscritos (X2) temos a tag <sub></sub>.
